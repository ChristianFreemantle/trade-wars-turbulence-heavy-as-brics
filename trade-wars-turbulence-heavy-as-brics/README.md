# Trade Wars & Turbulence: Heavy as BRICS
## Analyzing the Global Economic Impact of Modern Trade Conflicts on Major & Emerging Economies

### Overview
This project investigates the economic impact of global trade wars, focusing on:
- U.S.–China trade tensions
- The evolving role of BRICS nations (Brazil, Russia, India, China, South Africa)
- Volatility in emerging markets like South Africa

### Goals
- Visualize key financial indicators over time
- Analyze GDP, currency, inflation, and commodity prices
- Compare responses of major vs. emerging economies
- Forecast future trends using machine learning models

### Tools
- Python
- Pandas, Jupyter
- SQL (BigQuery)
- Tableau 
- GitHub for version control

### How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run notebooks in the order provided
